Yang dibutuhkan
- xampp
- php
- mysql

Cara instalasi
- Ekstrak file Tugas2 ke file htdocs
- buka phpmyadmin kemudain bikin database baru dengan nama "contact"
- impor file contact.sql ke database

Cara menjalankan
- buka xampp
- kemudian nyalakan apache dan mysql
- salin link "http://localhost/Tugas2/" ke browser 
- siap digunakan
